# Social Media Downloader
Telegram bot for downloading media from Social Media.

## Usage
1. Clone repository
2. Define BOT_TOKEN via environment. Define USER_ID as a list of users allowed to use it (comma-separated). Define RAPID_API_KEY to download Instagram videos.
3. Run `social-media-downloader`.

Or use a Docker:
`docker pull ghcr.io/toshik1978/social-media-downloader:latest`

Now you can use commands `/start`, `/help`, `/stats`, `/resetstats` or send Twitter/Instagram/YouTube links
