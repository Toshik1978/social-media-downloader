# Social Media Downloader
Telegram bot for downloading media from Social Media.

## Usage
1. Clone repository
2. Fill [config.py](https://gitlab.thedatron.io/anton/social-media-downloader/blob/main/config.py) with your data. Or use environment!
3. Install dependencies with `make code.deps`
4. Start the bot with `make app.run`
5. Now you can use commands `/start`, `/help`, `/stats`, `/resetstats` or send Twitter/Instagram/YouTube links
